Vector Database Endpoints

GET /vectordb/weaviate Get Weaviate status

Check the operational status of the Weaviate vector database service

curl -X GET "https://api.vectordbcloud.com/vectordb/weaviate" \ -H "X-API-Key: your-api-key-here"
POST /vectordb/weaviate Query Weaviate

Execute a GraphQL query against the Weaviate vector database

curl -X POST "https://api.vectordbcloud.com/vectordb/weaviate" \ -H "X-API-Key: your-api-key-here" \ -H "Content-Type: application/json" \ -d '{ "query": { "Get": { "ClassName": { "field1": "value1", "field2": "value2" } } } }'

Execute a vector similarity search against the Qdrant database

curl -X POST "https://api.vectordbcloud.com/vectordb/qdrant" \ -H "X-API-Key: your-api-key-here" \ -H "Content-Type: application/json" \ -d '{ "collection_name": "my_collection", "query": { "vector": [0.1, 0.2, 0.3, 0.4], "limit": 10, "offset": 0, "with_payload": true, "with_vector": false } }'
